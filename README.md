# Hide Your Armor

My **first** ever minecraft **mod**.

A simple Fabric mod that allows you to **toggle armor visibility** with a keybind. Perfect for screenshots or when you want to see your skin without removing armor.

## Features
- Toggle armor visibility with **'H'** key (configurable)
- Only affects your own armor visibility (Hopefully)
- **_Compatible with Wildfire's Gender Mod_**
- **Client-side** only, works on servers

## **Dependencies**
- Fabric API
- Minecraft 1.21.4
- Java 21
- Optional: Wildfire's Gender Mod (for breast armor compatibility)

## Installation
1. Install Fabric Loader
2. Make sure to have Fabric API installed
3. Place mod in your mods folder
4. Run the game

## Usage
- **Press 'H'** to toggle armor visibility
